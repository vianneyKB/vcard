  • latest update: 07/31/2020
  • by: mutationthemes
  • email: ghribbenaissa@gmail.com

Description

FOXEVCARD is a modern and premium v-card, cv, resume template made using the very popular neumorphism flat design in a beautiful manner, it allows you to introduce yourself with the most important informations to your futur clients or employers with a unique and a professional way.

The template is responsive that’s allows it to view it in the mobile devices also.

All the html ,css, javascript code are well organized and commented to make any change easy to do without any problems.

Features

  • Premium design
  • Retina ready
  • Parallax animation
  • Working contact from
  • Built using bootstrap
  • Light and Dark mode
  • Multiple color choices built in
  • Project single page
  • Blog pages
  • Tabed navigation
  • Filterabe portfolio

 

Theme installation

When you download the main download file you will  find several folder  :

  •       HTML folder this is the folder that contain all the files of the template
  •       LICENSES folder this folder contain the licenses of the elements used.
  •       DOCUMENTATION : this folder contain the documentation of the theme please read the documentation carefully for further information

File Structure

In this section a guide to understanding how FOXEVCARD is structured.

CSS subfolder 

This folder contain all the css files that are required by the template those files are :

  1. main.css
  2. bootstrap.css
  3. venobox.css
  4. font.css
  5. owl carousel subfolder that style the carousel testimonials
  6. animate.css
  7. switcher that style the styler for demo purposes

JS subfolder

This folder contain all the js files required by the template those files are :

  1. jquery
  2. bootstrap
  3. venobox
  4. owl carousel
  5. smooth-scroll.js
  6. morphext
  7. parallax
  8. easytabs
  9. placeholders
  10. easing
  11. switcher
  12. script

ASSETS subfolder

This subfolder contain other subfolders :  the img subfolder and the svg subfolder, evenif those are not included in the main download files and replaced with placeholders, but you can replace theme easly with your own assets.

  • Images used in html pages should be found in /assets/img.
  • Svg used in html pages should be found in /assets/svg.

 

WEBFONTS subfolder

This folder contain all the webfonts from fontawsome used in this template, you can use a wide range of icons that suits your need and the . For more info on how to setup and use fontawsome please visit this link below :

https://fontawesome.com/how-to-use/on-the-web/setup/getting-started?using=web-fonts-with-css

 

 

Edit the Color Scheme

You can easly change the color sheme of the theme weither for the light or the dark version by just using the styler included with the template.

The styler component gives the user the ability to change the color of the theme by loading a specific css style file for that specific color.

All the style are located in the css folder in the styles folder.

Plugins

An overview of the plugins supplied with FOXEVCARD

 

Jquery

jQuery is a JavaScript library designed to simplify the client-side scripting of HTML. It is free, open-source software using the permissive MIT License.

 

Bootstrap

A full javascript source for the version of bootstrap used in this  templatte. FOXERESTO ships with Bootstrap 4.1.3. This will be updated as further Bootstrap versions are released.

 

Smooth Scroll

A lightweight script to animate scrolling to anchor links. This provides a better user experience for navigating one-page sites or long landing pages.

 

Venobox

Just another responsive jQuery Lightbox plugin, suitable for images, inline contents, iFrames, Google-Maps, Ajax requests, Vimeo and YouTube videos.

 

Parallax

Simple, lightweight Parallax Engine that reacts to the orientation of a smart device

 

Owl carousel

Touch enabled jQuery plugin that lets you create a beautiful responsive carousel slider.

Morphext

A simple, high-performance and cross-browser jQuery rotating / carousel plugin for text phrases powered by Animate.css.

 

Easytabs

EasyTabs is a lightweight jQuery plugin to provide full tab functionality, while leaving the styling up to you.

 

Easing

Uses the built in easing capabilities added In jQuery  to offer multiple easing options

 

Placeholders

Placeholders.js is a polyfill (or shim, or whatever you like to call it) for the HTML5 placeholder attribute, as defined in the HTML5 specification. Placeholder attributes are valid on input elements of various types and textarea elements.

 

Switcher

Contains  additional scripts that setup the color theme styler

Contains  additional scripts required by FOXEYOGA.
